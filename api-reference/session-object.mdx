---
title: "The session object"
---

Represents a session in a project.

---

## Attributes

<ResponseField name="id" type="string">
  A unique ID for the session.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the session, which should reflect its core content for easy identification.
</ResponseField>

<ResponseField name="output_language" type="enum<string>">
  The language in which the output is generated. For example, if `output_language` is `EN`, the output generated by Powerdrill is in `English`.
</ResponseField>

<ResponseField name="job_mode" type="enum<string>">
  The job mode set for the session. 
</ResponseField>
